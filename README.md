# ML-LogisticRegressor
**Project Title: Heart Attack Probability Prediction using Logistic Regression**

This project focuses on predicting the probability of a heart attack using a healthcare dataset and a Logistic Regression model. The dataset contains various medical and lifestyle-related features such as age, cholesterol level, blood pressure, heart rate, and other relevant health indicators.

The primary objective of this project is to build a classification model that can accurately identify whether a person is at risk of a heart attack or not. Logistic Regression was chosen due to its simplicity, interpretability, and effectiveness for binary classification problems.

Before training the model, data preprocessing steps were applied, including handling missing values, feature scaling, and encoding categorical variables where necessary. The dataset was then split into training and testing sets to evaluate model performance.

After training, the model achieved an accuracy of approximately **88%**, indicating a reasonably good performance in predicting heart attack probability. This suggests that the model can correctly classify a significant portion of cases, although there is still room for improvement.

Further improvements can be made by:

* Using advanced models like Random Forest or Gradient Boosting
* Performing feature engineering
* Applying hyperparameter tuning
* Using cross-validation for better generalization

This project demonstrates the practical application of machine learning in the healthcare domain, where predictive models can assist in early risk detection and support medical decision-making.
